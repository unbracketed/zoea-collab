## Version Management

All tool versions are managed via **mise** in `.mise.toml`:

```toml
[tools]
python = "3.12.8"
node = "24"
```

Python dependencies are locked in:
- `backend/pyproject.toml` - Dependency specifications
- `backend/uv.lock` - Locked versions (generated by uv)

Node dependencies are locked in:
- `frontend/package.json` - Dependency specifications
- `frontend/package-lock.json` - Locked versions (generated by npm)

## System Requirements

### Development

- **Operating System:** macOS, Linux, or Windows (WSL2)
- **Memory:** 4GB RAM minimum, 8GB recommended
- **Disk Space:** 2GB for dependencies and virtual environments
- **Python:** 3.12+ (managed by mise)
- **Node.js:** 24+ (managed by mise)

### Production

- **Operating System:** Linux (Ubuntu 20.04+ or similar)
- **Memory:** 8GB RAM minimum, 16GB recommended for AI operations
- **CPU:** 2+ cores
- **Disk Space:** 20GB+ depending on document storage
- **Database:** PostgreSQL 12+ or later
- **Python:** 3.12+
- **Node.js:** 24+ (for build process)

## Dependency Installation

### Backend

```bash
cd backend
uv sync                    # Install all dependencies
uv sync --group dev       # Install with dev dependencies
```

### Frontend

```bash
cd frontend
npm install               # Install all dependencies
npm ci                    # Clean install (CI environments)
```

### Both (via mise)

```bash
mise run install          # Install backend + frontend
```

## Updating Dependencies

### Backend

```bash
cd backend
uv add package-name@version      # Add or update specific package
uv lock                          # Update lock file
```

### Frontend

```bash
cd frontend
npm install package-name@version # Update specific package
npm update                       # Update all packages
```
